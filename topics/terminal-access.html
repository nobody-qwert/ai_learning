Terminal Access

Command-line interfaces and tools for interacting with AI models and services.

OpenAI CLI

# Install OpenAI CLI pip install openai # Set API key export OPENAI_API_KEY="your-api-key-here" # Make API calls openai api completions.create \ -m text-davinci-003 \ -p "Explain quantum computing" \ --max-tokens 100

Ollama

Run large language models locally via command line:

# Install and run Llama 2 ollama run llama2 # Chat with the model >>> Hello! How can I help you today? # Run specific model variants ollama run codellama:7b

Curl API Requests

Direct API access for automation and scripting:

curl -X POST https://api.openai.com/v1/chat/completions \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $OPENAI_API_KEY" \ -d '{ "model": "gpt-3.5-turbo", "messages": [{"role": "user", "content": "Hello!"}] }'
